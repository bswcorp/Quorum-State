# Quorum-State Branding Guidelines

Dokumen ini mendefinisikan identitas visual resmi untuk ekosistem **Quorum-State ($QSTATE)**.

## 1. Logo Resmi
Logo menggabungkan simbol "Q" futuristik dengan pola 676 node yang saling terhubung, melambangkan konsensus Quorum dan status superposisi kuantum.

- **Logo Utama:** `docs/assets/logo.png`
- **Konsep:** Kesatuan, Keamanan Kuantum, dan Kecerdasan Buatan.

## 2. Palet Warna (Color Palette)
Gunakan kode warna berikut untuk menjaga konsistensi pada UI/UX dan materi pemasaran:
- **Quantum Purple:** `#9D50BB` (Melambangkan inovasi dan AI)
- **Deep Space Blue:** `#000428` (Melambangkan keamanan dan stabilitas)
- **Electric Cyan:** `#00D2FF` (Melambangkan kecepatan transaksi/TPS)

## 3. Tipografi
- **Headlines:** Montserrat atau Orbitron (Kesan futuristik dan berani).
- **Body Text:** Inter atau Roboto (Kesan bersih dan mudah dibaca).

## 4. Slogan Resmi
> *"The Last Safe Haven Before Q-Day."*
> *"Where Quorum Meets Quantum."*

## 5. Penggunaan Logo
- Pastikan ada "ruang bernapas" di sekitar logo.
- Jangan mengubah proporsi logo secara asimetris.
- Gunakan latar belakang gelap untuk menonjolkan efek cahaya neon pada logo.
