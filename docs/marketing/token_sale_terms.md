## 2. Jadwal Vesting (Penguncian Token)
Untuk menjaga stabilitas harga saat listing di bursa, token investor awal akan didistribusikan secara bertahap:
*   **TGE (Token Generation Event):** 20% token dapat diakses langsung.
*   **Cliff:** 3 bulan penguncian penuh setelah TGE.
*   **Linear Vesting:** Sisa 80% token akan dilepaskan secara linear setiap bulan selama 12 bulan berikutnya.
